Taiwan Clinical Trial Portal

Search Author Contact by Rank



Press the button on top of this page to get started!
Depressing the Search author contact by rank bring up a table with five columns . Clicking on the Count link results in navigating to the NIH clinical trials database classic web site showing all the clinical trials performed by the researcher(s). It is sorted by the number of clinical trials listed researcher(s) have done. Clicking the Contact Name link navigates the user to the Pub Med site where the publications of the selected researcher are shown. The user may find the contact information in the remaining columns. When the user clicks on the Search research button will navigate user to a separate table showing Rank, NCT ID, Brief Title, Condition and Investigator at the top of the table will be the number of records found in the NIH clinical trial data base for that specific disease, the API query interface will also be shown. Once the separate table is displayed the user can click on the NCT ID to navigate to the NIH clinical trial database the specific record for that NCT ID will appear. Clicking on individual Investigator, the contact information will be displayed in a pop up window. After each search is necessary to clear the Type study field before beginning the next search.
Fill in the Type study field and then press the Search button to initiate the search.
