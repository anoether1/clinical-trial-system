The Search research button allows the researcher to search for research information by number of records starting from the minimum rank research clinical trial to the maximum rank research clinical trial. The Search author contact by rank button searches by the name of the researchers conducting the study, It lists the authors sorted by number of clinical trials performed. It assigns a rank based on the number of studies the researcher(s) have performed. The Search by Clinical Trial Phase button allows the user to search by phase or phases of clinical trial in a given therapeutic area.
This page requires the user to enter a therapeutic area and a starting rank and ending rank, and then hit the search button e.g. renal disease in the "Type what you want to search for" box, 1 in the Minimum rank box and 10 in the Maximum rank box then hit Search. The results will be a list of records of clinical trial studies. After each search is necessary to clear the Type study field before beginning the next search.
Depressing the Search author contact by rank brings up a table with five columns. Clicking on the Count link results in navigating to the NIH database classic web site showing all the clinical trials performed by the researcher(s). It is sorted by the number of clinical trials listed researcher(s) have done. Clicking the Contact Name link navigates the user to the Pub Med site where3 the publications of the selected researcher are shown. The user may find the contact information in the remaining columns. After each search is necessary to clear the Type study field before beginning the next search.
The Search by Clinical Trial Phase button results of the clinical phase or phases results in navigating to the NIH database page showing the studies done in Taiwan in the selected phase.
